    Images

    To add an image you simply add the image as you would in native HTML

    If you'd like to change the dimensions of the image you can add the mdp-width="" and mdp-height="" property, if you use just one of theese properties the other adjusts automaticaly. If you add both of them the image will stretch to fit the area.

    Special attributes

    MDP offers some attributes that aren't commonly used, here they are:

    mdp-limit-computer-width

    mdp-limit-computer-width limits the max width of the body to 1000px so text can still be easily read on big screens where the distance from the end of the line to the start of the other line are too large to easily track. This will only works if the attribute is set to the body element

    mdp-hide-on-small-screens

    mdp-hide-on-small-screens hides the element on screens smaller than 800 pixels.

    mdp-hide-on-large-screens

    mdp-hide-on-large-screens hides the element on screens larger than 800 pixels.

    mdp-show-loading-screen

    mdp-show-loading-screen can be added to the body element, it shows a loading screen while the content of the page is being loaded.
